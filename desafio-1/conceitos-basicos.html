💻 História da Computação

A evolução dos computadores passou por quatro gerações principais:

  1. 1ª Geração (1940-1956): Válvulas termiônicas e grandes dimensões
  2. 2ª Geração (1956-1963): Transistores e redução de tamanho
  3. 3ª Geração (1964-1971): Circuitos integrados e maior eficiência
  4. 4ª Geração (1971-atual): Microprocessadores e computadores pessoais

O ENIAC, criado em 1946, pesava incríveis 30 toneladas e ocupava 167m²!

🖥️ Client-Server Architecture

Na arquitetura web, temos dois componentes principais:

📱 Clients (Clientes)

São os dispositivos ou aplicativos que solicitam recursos de um servidor. Exemplos:

  • Navegadores web (Chrome, Firefox, Safari)
  • Aplicativos móveis
  • Programas de email

🖥️ Servers (Servidores)

São sistemas que fornecem serviços e recursos para os clients. Principais tipos:

  • Web Servers: Servem páginas HTML
  • Database Servers: Armazenam e gerenciam dados
  • File Servers: Compartilham arquivos
  • Mail Servers: Gerenciam emails
"A comunicação entre client e server segue o modelo request-response: o client pede, o server responde."
