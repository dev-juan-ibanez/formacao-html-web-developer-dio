📄 Anatomia de um Documento HTML

Todo documento HTML válido possui uma estrutura básica:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <!-- Conteúdo visível -->
</body>
</html>

🔍 Partes Essenciais:

  • <!DOCTYPE html> - Declaração do tipo de documento
  • <html> - Elemento raiz do documento
  • <head> - Metadados e informações da página
  • <body> - Conteúdo visível para o usuário

📋 Trabalhando com Listas

• Listas Não Ordenadas (<ul>)

Usadas quando a ordem não importa não é importante:

  • Item sem ordem específica
  • Lista de compras
  • Características

1. Listas Ordenadas (<ol>)

Usadas para itens em sequência importante:

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

🎨 Formatação de Texto

HTML oferece diversas tags para destacar e formatar conteúdo textual:

Texto importante | Texto em itálico | Texto sublinhado | Texto marcado

Textosubscrito | Textosobrescrito | Texto pequeno | Texto removido

"HTML é como o esqueleto de uma página web - define a estrutura básica sobre a qual todo o resto é construído."
